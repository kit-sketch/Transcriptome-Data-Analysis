Although SERINC5 expression appears much higher in high-infectivity cell lines compared to low-infectivity ones, the difference was not statistically significant (Welch t-test, p = 0.104), likely due to small sample size and high variability.

A one-way ANOVA revealed a highly significant difference in SERINC5 expression across infectivity groups (F = 60.55, p < 0.001), indicating that expression varies systematically with infectivity level.

Tukey’s post-hoc analysis revealed significant differences between all infectivity groups. SERINC5 expression was highest in high-infectivity cell lines, intermediate in medium-infectivity lines, and lowest in low-infectivity lines (all pairwise comparisons p < 0.05), indicating a graded increase in expression with infectivity

Earlier we saw: t-test (Low vs High) → not significant ANOVA → significant Tukey → ALL pairs significant This is because Tukey uses pooled variance from ANOVA and is More

## -------- STEP 2: ALIGN WITH INFECTIVITY --------
############################################

cell_order <- infect$CellLine

# Ensure all names exist
common_cells <- intersect(cell_order, colnames(expr_merged))

expr_mat <- expr_merged[, common_cells]

# Match infect order
infect_sub <- infect[infect$CellLine %in% common_cells, ]
expr_mat <- expr_mat[, infect_sub$CellLine]

expr_mat <- apply(expr_mat, 2, as.numeric)

############################################
## -------- STEP 3: GENE-WISE CORRELATION --------
############################################

cor_results <- apply(expr_mat, 1, function(gene_expr) {

  valid <- is.finite(gene_expr) & is.finite(infect_sub$Infectivity)

  if (sum(valid) < 3) {
    return(c(r = NA, R2 = NA, p = NA))
  }

  ct <- cor.test(
    gene_expr[valid],
    infect_sub$Infectivity[valid],
    method = "pearson"
  )

  c(
    r  = unname(ct$estimate),
    R2 = unname(ct$estimate)^2,
    p  = ct$p.value
  )
})

############################################
## -------- STEP 4: RESULTS TABLE --------
############################################

cor_df <- data.frame(
  GeneID = expr_merged$GeneID,
  t(cor_results),
  row.names = NULL
)

cor_df$r  <- as.numeric(cor_df$r)
cor_df$R2 <- as.numeric(cor_df$R2)
cor_df$p  <- as.numeric(cor_df$p)

############################################
## -------- STEP 5: MULTIPLE TEST CORRECTION --------
############################################

cor_df$FDR <- p.adjust(cor_df$p, method = "BH")
